# Walmart-sales-performance-tableau

https://public.tableau.com/authoring/walmart_17726483589840/Sheet1/Dashboard%201#1
Built a Tableau dashboard analyzing Walmart weekly sales to identify seasonality, store performance patterns, and holiday uplift using cleaned retail data.
Cleaned and standardized date formats and numeric fields in Python, engineering time features (Year/Month/Week) to support trend analysis and visualization.
Delivered insights and recommendations to support demand planning and operational decision-making.
